# Icons

This folder should contain the extension icons.

For now, you can use simple placeholder images or create proper icons later.

Required files:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

You can create simple colored squares as placeholders using any image editor, or use online icon generators.

Suggested color: #FFC107 (yellow/gold matching the Verifeed brand)
